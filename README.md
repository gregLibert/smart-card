# smart-card

`smart-card` is a Go library developed to help understand the **ISO 7816** and **EMV** standards.

The project focuses on providing transparency in the exchanges between cards and terminals to make their communication more intelligible.

## Contributions

Contributions are welcome, especially those related to the bank card ecosystem. Submissions should respect the following requirements:

* **Scope:** Topics specifically related to payment cards and their protocols.
* **Quality:** Compliance with the project linter and maintenance of test coverage.
* **Clarity:** A strong emphasis on code readability and clear output formatting to help users understand the underlying communication.
